List Edit Mode - Feature Test

Test the new keyboard navigation edit mode in List items. Use the focus tracker at the bottom to monitor which element has focus and the current edit mode state.

Test 1: F2 enters edit mode from item level

Steps: Focus "Laptop" item → press F2
Expected: Focus moves to the delete button. Edit mode indicator shows ON.

Test 2: F2 exits edit mode back to item level

Steps: From Test 1 result (delete button focused) → press F2 again
Expected: Focus returns to the list item root. Edit mode indicator shows OFF.

Test 3: Tab in navigation mode exits the list

Steps: Focus "Laptop" item (navigation mode) → press Tab
Expected: Focus moves to the "After" button below the list (exits the list). Delete button is NOT focused.

Test 4: Tab in edit mode flows to next item's elements

Steps: Focus "Laptop" → F2 (enters edit mode, focus on delete btn) → Tab
Expected: Focus moves to "Monitor" item's delete button. Then Tab again moves to "Keyboard" item's delete button.

Test 5: Tab from last item in edit mode exits the list

Steps: Continue from Test 4 (focus on "Keyboard" delete button in edit mode) → Tab
Expected: Focus moves to the "After" button. Edit mode is cleared.

Test 6: Shift+Tab in edit mode flows backwards

Steps: Focus "Monitor" → F2 → Shift+Tab
Expected: Focus moves to "Laptop" item's delete button (last focusable element of previous item).

Test 7: Shift+Tab from first item exits the list backwards

Steps: Focus "Laptop" → F2 (on delete button) → Shift+Tab
Expected: Focus moves to "Before" button above the list.

Test 8: F7 toggles edit mode

Steps: Focus "Laptop" item → press F7
Expected: Focus moves to internal element (delete button). Press F7 again → focus returns to item root.
