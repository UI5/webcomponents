Toolbar Keyboard Navigation - Manual Test Page

1. Basic Arrow Key Navigation

Click "First", then use ArrowRight/Left. Expected: focus moves between buttons. Wraps at edges.

2. Home / End Keys

Focus any item, press Home → first item. Press End → last item.

3. Tab Exit and Re-entry

Focus "Second" button, Tab out to input below. Shift+Tab back → should restore "Second".

4. Skip Separators and Spacers

ArrowRight from "Before" should skip separator and spacer, land on "After".

5. Disabled Items Skipped

ArrowRight from "Enabled 1" should skip "Disabled" and land on "Enabled 2".

6. ToolbarSelect Key Delegation

Focus Select, press ArrowDown → dropdown opens (not toolbar nav). Press Home when closed → toolbar jumps to first.

7. Radio Group (ToolbarItem)

ArrowRight navigates within radio group. At boundary (last radio), next ArrowRight exits to Checkbox.

8. Checkbox Group (ToolbarItem)

ArrowRight moves through checkboxes sequentially. At last checkbox, exits to next item.

9. Input Caret Boundary

Type text, move caret to end, press ArrowRight → exits to "Next". If caret mid-text, ArrowRight stays in input.

10. Overflow Popover (Tab navigation)

Items forced to overflow. Click overflow button, then use Tab to navigate within popover.

11. RTL Direction

In RTL, ArrowLeft moves forward (next item), ArrowRight moves backward (prev item).

12. ToolbarItem with Single Child Controls

Arrow navigation between generic wrapped controls (ComboBox, Switch, DatePicker).

13. Backward Entry into Groups

ArrowLeft from "After" should enter checkbox group at last item (CB 3).
