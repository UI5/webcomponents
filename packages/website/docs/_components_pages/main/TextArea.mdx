### Max Length
You can use the <b>maxlength</b> and <b>showExceededText</b> properties to define characters limit
and to indicates how many characters are remaining before the limit or exceeding after the limit.

<MaxLength />

### Exceeded Text
Use the <b>showExceededTextFocus</b> property to show the character counter only when the TextArea is focused
or when the character limit is exceeded. The counter is also always visible when
<b>valueState</b> is "Critical" or "Negative".

The application can control threshold-based coloring by setting <b>valueState</b> based on character count.

<ExceededText />
